# 练习与实践

本目录包含每节课的实践资料和参考答案。

## 如何使用

1. 先完成对应课程的学习
2. 独立完成练习题（不要先看答案！）
3. 核对参考答案，理解差异
4. 把关键知识点用自己的话写下来

## 推荐实践项目

### 项目1：搭建个人 OpenClaw 助手
- 安装 OpenClaw 并完成基础配置
- 配置 SOUL.md 定义个性
- 接入至少一个渠道（推荐 Telegram）
- 安装 3-5 个常用 Skill

### 项目2：开发自定义 Skill
- 理解 SKILL.md 格式
- 编写一个简单的信息查询 Skill
- 发布到 ClawHub

### 项目3：模拟面试录音
- 录制自己的项目介绍（3分钟版）
- 找朋友做模拟面试
- 回看录音，找出不足
