## 风格说明

- **角色**：蓝色圆头机器猫（导师角色）+ 好奇少年（学生角色）
- **画风**：日系可爱清新漫画风，色彩明亮
- **用途**：穿插在课程 Markdown 文档中，辅助理解概念
- **引用方式**：`![漫画描述](../comics/comic-XX-xxx.png)`
